Mengenal Link dan Gambar di HTML

Link (Anchor Tag)

Untuk informasi tambahan bisa klik link berikut Google

Anda juga bisa membaca berita di Detik.com (link ini akan terbuka di tab baru).

Nonton youtube bisa klik link berikut Youtube.

Gambar (Image Tag)

Gambar dari Internet (URL)

Gambar dari File Lokal

Di bawah ini seharusnya ada gambar kucing (jika Anda punya filenya).

Menggabungkan Link dan Gambar

Anda juga bisa membuat sebuah gambar menjadi link dengan membungkus tag <img> dengan tag <a>.

Perbedaan tanpa adanya target="_blank" pada codingan. Berguna untuk membuka tab baru setelah meng-klik gambar tersebut.
