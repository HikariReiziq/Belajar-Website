Tag Tampilan Visual Makna Semantik (Semantic Meaning) Kapan Digunakan (When to Use)
<b> Tebal Tidak ada Hanya untuk menarik perhatian visual (nama, kata kunci).
<strong> Tebal Sangat Penting Untuk peringatan, konten krusial.
<i> Miring Suara/Teks Berbeda Untuk istilah asing, nama ilmiah.
<em> Miring Penekanan Untuk menekankan sebuah kata dalam kalimat.
