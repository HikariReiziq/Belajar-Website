Roadmap Belajar Web Development

1. Unorder List (<ul>)

Digunakan untuk daftar item di mana urutannya tidak penting. Secara default, akan ditandai dengan bulatan (bullet points).

  • Kopi
  • Teh
  • Susu

2. Ordered List (<ol>)

Digunakan untuk daftar item di mana urutannya PENTING. Secara default, akan ditandai dengan angka (1, 2, 3, ...).

  1. Bangun tidur
  2. Mandi
  3. Sarapan
  4. Berangkat kerja

3. Description List (<dl>)

Digunakan untuk menyajikan daftar istilah dan definisinya (seperti kamus atau glosarium).

HTML
Singkatan dari HyperText Markup Language, digunakan untuk membuat struktur halaman web.
CSS
Singkatan dari Cascading Style Sheets, digunakan untuk mengatur tampilan dan gaya halaman web.

Contoh Nested List (List Bersarang)

Anda bisa memasukkan list di dalam list lain. Ini sangat umum digunakan untuk membuat menu navigasi dropdown.

  • Makanan
